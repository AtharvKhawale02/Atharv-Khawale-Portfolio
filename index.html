Student Performance Prediction System

Python Django Scikit-Learn SQLite Chart.js

A comprehensive predictive analytics platform designed to forecast student academic performance based on various factors including previous grades, test scores, attendance records, and assignment submissions.

Key Features

  • Implemented Random Forest, Neural Networks, and Gradient Boosting algorithms for accurate performance modeling
  • Developed interactive dashboards for both students and faculty with real-time visualizations
  • Integrated with existing school management systems for seamless data flow
  • Role-based access control for different user types (students, teachers, administrators)

Technical Details

  • Backend: Django with REST API endpoints for data processing
  • Frontend: Custom dashboard with Chart.js for visualizations
  • Machine Learning: Scikit-Learn for model training and evaluation
  • Database: SQLite for development, with MySQL support for production
Impact

The system helps educators identify at-risk students early, allowing for timely interventions. Students can track their predicted performance and receive personalized recommendations for improvement.

AgriPredictAI

Streamlit Python Scikit-learn Pandas Matplotlib

An innovative machine learning solution designed to forecast prices of agricultural commodities, helping farmers and traders make informed decisions about planting, harvesting, and selling their produce.

Key Features

  • Utilized NHITS (Neural Hierarchical Interpolation for Time Series) model for accurate price forecasting
  • Collected and processed historical time-series data for multiple agricultural commodities (rice, pulses, tomatoes)
  • Developed interactive data dashboards for trend analysis and future price predictions
  • Implemented regression models to account for seasonal variations and market trends

Technical Details

  • Frontend: Streamlit for rapid prototyping and deployment
  • Data Processing: Pandas for cleaning and feature engineering
  • Visualization: Matplotlib and Plotly for interactive charts
  • Model Training: Scikit-learn for traditional models, custom NHITS implementation
Impact

AgriPredictAI helps farmers optimize their planting and selling strategies, potentially increasing their profits by 15-20%. Traders can use the forecasts to make better purchasing decisions, reducing market volatility.
